## Usage / Install

Clone repo, cd into folder and run:

```console
$ npm install
$ npm start
```

## Requirement

- npm

## Tools

- Create-react-app cli
- tachyons: CSS framework
- Robohash API
- JSONPlaceholder
- Redux
